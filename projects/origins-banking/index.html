Origins Banking

Origins Banking is a production-grade FiveM banking script built for serious roleplay servers — not a reskinned ATM menu. Multi-account management, peer transfers, savings, debt tracking, transaction logs and quick-contact shortcuts each live inside a cohesive NUI dashboard designed to feel like a physical in-world device.

Developed in partnership with Origins RP and NikX, the project pairs a dark cyber-industrial visual language with the mechanical, high-fidelity atmosphere Origins demands. Emerald for credits, crimson for debits, gold accents for hierarchy — every state is readable at a glance inside the heat of RP.

The interface was shaped as a tablet-first experience: sidebar navigation, balance cards, scrollable transaction feeds, frequent-transfer contacts and contextual actions that respond with subtle glow and motion. Behind the NUI, the Lua layer handles account persistence, server-side validation and framework integration so the UI never fights the simulation.

From dashboard telemetry to transfer confirmation, each screen was treated as product design — layout grids locked early, component states mapped for every edge case, then translated into a React NUI stack tuned for FiveM performance. The result is a banking system players actually want to open, not a chore they rush through.
